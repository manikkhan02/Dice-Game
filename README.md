# 🎲 Dice Game Project Readme 🎲

Welcome to the Dice Game project! This is a simple dice game implementation using HTML, CSS, and JavaScript.

## 📝 Overview
In this project, you'll find an interactive dice game where two players roll a dice each turn, and the player with the highest dice roll wins the round. The game consists of multiple rounds, and the player who wins the most rounds is declared the overall winner.

## 🎮 How to Play
1. Open the `index.html` file in your web browser.
2. Click on the "Roll Dice" button to roll the dice.
3. The dice will roll, and the results will be displayed.
4. The player with the highest dice roll wins the round.
5. Continue rolling until all rounds are completed.
6. The player with the most round wins is the overall winner.

## ✨ Features
- Simple and intuitive user interface.
- Random dice roll generator.
- Interactive gameplay for two players.
